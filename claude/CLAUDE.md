# Claude Code 設定

## IMPORTANT: エージェントルール

rules/agents.md のエージェント選択ルールに必ず従うこと。
サブエージェントには**目的（何をしたいか）だけを伝え、具体的なパスや実装手段は指定しない**。

## IMPORTANT: ナレッジ管理 (Obsidian CLI)

rules/obsidian-knowledge.md のナレッジ管理ルールに必ず従うこと。
タスク開始時の検索、メモの記録タイミング、タスク完了前のセルフチェックはすべて必須である。

## IMPORTANT: 課題作成テンプレート

Backlog 課題や GitHub Issue を作成する際は、rules/issue-template.md のテンプレートに必ず従うこと。

## IMPORTANT: コーディング規約

rules/coding-conventions.md のコーディング規約に必ず従うこと。
