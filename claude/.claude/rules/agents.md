# エージェントオーケストレーション

## Nuxt プロジェクト（必須）

`nuxt.config.ts` が存在する場合、**必ず** nuxt4-coder エージェントを使用すること。
直接 Edit/Write ツールで .vue や server/ 配下のファイルを編集してはならない。

判定フロー:
1. Glob で nuxt.config.ts を確認
2. 存在する場合 → Task ツールで nuxt4-coder を呼び出し
3. 存在しない場合 → 直接 Edit/Write を使用可

## コードレビュー（必須）

コード作成・変更後、**必ず** code-reviewer エージェントを実行すること。

## 並列実行

独立した操作には並列タスク実行を使用:
```
3つのエージェントを並列で起動:
1. auth.tsのセキュリティ分析
2. キャッシュシステムのパフォーマンスレビュー
3. utils.tsの型チェック
```
